Significado de las etiquetas principales
Etiqueta Significado
<!DOCTYPE html> El tipo de archivo y lenguaje que se utilizara
<html lang="es"> El lengiaje que sera utilizado en la pagina
<head> Es el encabezado de la pagina web donde se ponen los metadatos de la pagina web
<meta charset="UTF-8"> Utiliza el tipo de codificación UTF-8 ya que es el universal y el que todos los navegadores soportan
<meta http-equiv="X-UA-Compatible" content="IE=edge"> Se usa para dar compatibilidad al navegador edge
<meta name="viewport" content="width=device-width, initial-scale=1.0"> Hace que la pagina sea multiplataforma
<meta name="author" content="Naoko"> Se pone el nombre del autor
<title> Se incerta el titulo de la pagina
<link rel="stylesheet" href="/css/estilos.css"> Da relación al archivo HTML de donde y como obtner la información
